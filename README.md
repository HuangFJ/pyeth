Ethereum for Python
===============================

version number: 0.0.1
author: Jon Huang

Overview
--------

full implementation of the Ethereum protocol from scratch for python

Installation / Usage
--------------------

To install use pip:

    $ pip install pyeth


Or clone the repo:

    $ git clone https://github.com/HuangFJ/pyeth.git
    $ python setup.py install
